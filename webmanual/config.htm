Настройка

Если вы администратор, то для начала работы в MigApp рекомендуем предварительно выполнить следующие настройки окружения:
1. Скачайте и установите Microsoft SQL Server 2019 Express или выше, если у вас его нет.
1.1 Для установки Microsoft SQL Server 2022 скачайте установщик с официального сайта или по ссылке: https://download.microsoft.com/download/SQL2022-SSEI-Expr.exe.
1.2 Перед установкой ознакомьтесь с системными требованиями и лицензионным соглашением.
1.3 В процессе установки, пожалуйста, запустите файл SQL2022-SSEI-Expr.exe или иной, если вы используете другую версию Microsoft SQL Server. Откроется стартовое окно установщика (рис.6).
Рисунок 6 — Стартовое окно установщика SQL Server 2022 Express
1.4 Нажмите на кнопку "Базовая". В окне "Условия лицензии Microsoft SQL Server" (рис. 7) ознакомьтесь с условиями лицензии, после этого нажмите на кнопку "Принять".
Рисунок 7 — Окно установщика "Условия лицензии Microsoft SQL Server"
1.5 Если вы хотите изменить папку установки, в окне "Укажите целевое расположение SQL Server" (рис. 8) нажмите на кнопку "Обзор". В открывшемся проводнике (рис. 9) выберите папку, куда вы хотите установить сервер. Нажмите на кнопку "Установить". Иначе просто нажмите на кнопку "Установить".
Рисунок 8 — Окно установщика "Укажите целевое расположение SQL Server"
Рисунок 9 — Окно проводника
1.6 Начнётся процесс установки (рис. 10). Ожидайте завершения установки.
Рисунок 10 — Окно установщика в процессе установки
1.7 После завершения установки закройте окно установщика.
Рисунок 11 — Окно установщика "Установка завершена"
 
2. Скачайте и установите SQL Server Management Studio, если у вас его нет.
2.1 Для установки SQL Server Management Studio скачайте установщик с официального сайта или по ссылке: https://aka.ms/ssmsfullsetup?clcid=0x419.
2.2 Перед установкой ознакомьтесь с системными требованиями и лицензионным соглашением.
2.3 В процессе установки, пожалуйста, запустите файл SSMS-Setup-RUS.exe или иной, если вы используете другую версию SQL Server Management Studio. Откроется стартовое окно установщика (рис.12).
Рисунок 12 — Стартовое окно установщика SSMS
2.4 Если вы хотите изменить папку установки приложения, нажмите кнопку "Изменить" в стартовом окне установщика (рис. 12). В открывшемся проводнике (рис. 13) выберите папку, в которую вы хотите установить приложение. Нажмите на кнопку "Установить". Иначе просто нажмите на кнопку "Установить".
Рисунок 13 — Окно проводника
2.5 Начнётся процесс установки (рис. 14). Ожидайте завершения установки.
Рисунок 14 — Окно установщика в процессе установки
2.6 После завершения установки нажмите кнопку "Закрыть".
Рисунок 15 — Окно установщика "Установка завершена"
3. Запустите SQL Server Management Studio и произведите установку базы данных.
3.1 При запуске SSMS вас встретит окно "Соединение с сервером" (рис. 16). Если в поле "Имя сервера" отсутствует имя сервера, раскройте выпадающий список, если в нём тоже отсутствует имя сервера, нажмите "Обзор". Иначе выберите имя сервера и нажмите на кнопку "Соединить" и перейдите к пункту 3.3.
Рисунок 16 — Окно SSMS "Соединение с сервером"
3.2 В окне "Выбор серверов" (рис. 17) раскройте пункт "Ядро СУБД" и выберите нужный сервер и нажмите на кнопку "ОК".
Рисунок 17 — Окно SSMS "Выбор серверов"
3.3 В окне "Соединение с сервером" (рис. 16) нажмите кнопку "Соединить". После подключения к серверу вы увидите главное окно SSMS (рис. 18) Если у вас возникла проблема с подключением, перейдите к главе Устранение типовых проблем.
Рисунок 18 — Главное окно SSMS
3.4 В левой части окна находится обозреватель объектов (рис. 19). Нажмите правой кнопкой мыши по объекту "Базы данных", в открывшемся контекстном меню выберите пункт "Импорт приложения уровня данных...".
Рисунок 19 — Обозреватель объектов
3.5 В открывшемся стартовом окне (рис. 20) нажмите на кнопку "Далее".
Рисунок 20 — Стартовое окно импорта
3.6 Во вкладке "Параметры импорта" (рис. 21) нажмите на кнопку "Обзор". В открывшемся проводнике (рис. 22) найдите папку куда вы извлекли архив с установщиком MigApp и выберите файл MigDB_Preset.bacpac. Нажмите на кнопку "Далее".
Рисунок 21 — Окно импорта "Параметры импорта"
Рисунок 22 — Окно проводника
3.7 Во вкладке "Параметры базы данных" (рис. 23), в поле "Имя новой базы данных", вы можете изменить название базы данных. Нажмите на кнопку "Далее".
Рисунок 23 — Окно импорта "Параметры базы данных"
3.8 Во вкладке "Сводка" (рис. 24) нажмите на кнопку "Готово".
Рисунок 24 — Окно импорта "Сводка"
3.9 Начнётся процесс импорта (рис. 25). Ожидайте завершения импорта.
Рисунок 25 — Окно импорта в процессе работы
3.10 После завершения импорта нажмите на кнопку "Закрыть".
Рисунок 26 — Окно импорта завершило процесс импорта
При настройке необходимо учесть, что все действия предназначены для администратора и производятся только на сервере.
Если возникли проблемы при настройке окружения перейдите в главу Устранение типовых проблем или в главу Частые вопросы.
