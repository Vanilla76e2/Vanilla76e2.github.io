Основные понятия и термины

Перед началом работы в MigApp рекомендуем ознакомиться с основными понятиями и терминами:
Приложение — это компьютерная программа, предназначенная для оказания помощи людям в выполнении какой-либо деятельности.
База данных — совокупность данных, хранимых в соответствии со схемой данных, манипулирование которыми выполняют в соответствии с правилами средств моделирования данных.
SQL — декларативный язык программирования, применяемый для создания, модификации и управления данными в реляционной базе данных, управляемой соответствующей системой управления базами данных.
Microsoft SQL Server - это система управления реляционными базами данных, разработанная компанией Microsoft. Как сервер баз данных, это программный продукт с основной функцией хранения и извлечения данных по запросу других программных приложений, которые могут работать либо на том же компьютере, либо на другом компьютере по сети (включая Интернет).
SQL Server Management Studio (SSMS) — утилита из Microsoft SQL Server 2005 и более поздних версий для конфигурирования, управления и администрирования всех компонентов Microsoft SQL Server.
Логин — это имя пользователя, которое необходимо для входа в информационные системы, приложения, аккаунты.
Пароль — это условное слово или произвольный набор знаков, состоящий из букв, цифр и других символов, предназначенный для подтверждения личности или полномочий.
Установщик (англ. installer) — это программа, которая устанавливает программное обеспечение на компьютер.
Проводник — это программа, которая позволяет перемещаться между папками и дисками на компьютере.
Сервер — сетевой компьютер, обрабатывающий запросы от других компьютеров в локальной или глобальной сети.
Функциональность программного обеспечения — набор возможностей (функций), которые предоставляет данная система.
Накопитель информации — устройство для хранения информации. Используется в персональных компьютерах, ноутбуках и т.д.
Операционная система (ОС) —  программное обеспечение, управляющее компьютерами и позволяющее запускать на них прикладные программы.
Оперативная память (ОЗУ) — часть системы компьютерной памяти, в которой во время работы компьютера хранится выполняемый машинный код (программы), а также входные, выходные и промежуточные данные, обрабатываемые процессором.
Процессор — это основной вычислительный элемент компьютера, от которого во многом зависит скорость его работы и производительность.
Материнская плата — это сложная многослойная печатная плата, на которой устанавливаются основные компоненты персонального компьютера.
IP-адрес — это уникальный идентификатор, который присваивается устройству в интернете или локальной сети.
Комплектующее изделие — это составная часть финального изделия, которая является конструктивно законченным стандартным или иным изделием (например, прибор, блок, модуль, узел и т.п.).
Организационная техника (оргтехника) — это техническое оборудование, обеспечивающее работу любого офиса вне зависимости от вида деятельности компании.
Учётная запись — это хранимая в компьютерной системе совокупность данных о пользователе, необходимая для его опознавания (аутентификации) и предоставления доступа к его личным данным и настройкам.
Пользовательские таблицы — в данном проекте, это таблицы, к которым имеют доступ пользователи. В перечень пользовательских таблиц входят: сотрудники, компьютеры, ноутбуки, планшеты, оргтехника, мониторы.
