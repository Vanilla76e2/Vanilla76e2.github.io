Описание окон приложения

Окно авторизации программы MigApp (рис. 32) имеет следующую функциональность:
  • Поле для ввода пароля;
  • Поле для ввода логина;
  • Кнопка закрытия окна;
  • Кнопка открытия руководства;
  • Флажок "Запомнить пароль";
  • Кнопка "Вход"
  • Кнопка обновления подключения к базе данных;
  • Кнопка открытия окна настроек;
  • Кнопка "Показать пароль".
Рисунок 32 — Окно авторизации
Окно настроек программы MigApp (рис. 33) имеет следующую функциональность:

  • Поле для ввода названия базы данных;
  • Кнопка подтверждения настроек;
  • Кнопка закрытия окна.
